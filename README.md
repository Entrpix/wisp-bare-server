# Wisp Bare Server
Run Wisp and Bare w/ One File (SSL)

## Installation
```sh
$ git clone https://github.com/entrpix/wisp-bare-server
$ cd wisp-bare-server
$ npm i
# Change SSL Certs
$ npm start
```
